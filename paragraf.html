Apa itu HTML dan CSS?

HTML (HyperText Markup Language) adalah bahasa markup standar untuk membuat halaman web. HTML mendefinisikan struktur dan konten sebuah halaman web dengan menggunakan elemen-elemen atau tag-tag tertentu.

CSS (Cascading Style Sheets) adalah bahasa stylesheet yang digunakan untuk mengatur tampilan atau presentasi dari dokumen HTML. Dengan CSS, Anda dapat mengontrol tata letak, warna, font, dan berbagai aspek visual lainnya dari sebuah halaman web.

HTML Tag

HTML menggunakan tag-tag untuk mendefinisikan elemen-elemen atau bagian-bagian dari sebuah halaman web. Tag-tag ini memberikan struktur dan konten pada halaman web. Beberapa contoh tag HTML yang umum digunakan antara lain:

  • <html> - Mendefinisikan dokumen HTML
  • <head> - Mendefinisikan informasi metadata dari dokumen HTML
  • <title> - Mendefinisikan judul dari dokumen HTML
  • <body> - Mendefinisikan isi dari dokumen HTML
  • <h1> - Mendefinisikan heading level 1
  • <p> - Mendefinisikan sebuah paragraf
  • <a> - Mendefinisikan sebuah hyperlink
  • <img> - Mendefinisikan sebuah gambar
